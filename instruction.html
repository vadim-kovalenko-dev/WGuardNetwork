Как подключить VPN

Шаг 1. Получите ключ в Telegram

1. В чате нажмите START (или «Начать»).

2. Выберите в меню: ➕ WireGuard → Тип вашего устройства.

3. Введите любое название для ключа (например, MyDevice) и нажмите на кнопку «👤Это для меня».

4. Перейдите в раздел «Мои конфиги» и скачайте файл настроек (или используйте QR-код для телефона).

Шаг 2. Настройте приложение WireGuard

1. Установите приложение: Скачайте WireGuard для вашего устройства (доступно в любом магазине приложений).

2. Добавьте ключ:

• Откройте WireGuard и нажмите кнопку «Добавить туннель» (или иконку +).

• Выберите «Импорт из файла» и укажите скачанный в боте файл.

• Для смартфонов: можно выбрать «Сканировать QR-код».

3. Включите: Нажмите кнопку «Активировать» или переведите ползунок в рабочее состояние.
